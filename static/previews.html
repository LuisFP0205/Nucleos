Previews dos Overlays

🎨 Temas de Overlay
Chat
Mensagens em tempo real via WebSocket. Suporta emotes customizados da Twitch, Kick e YouTube. Novas mensagens entram pelo topo e somem após o tempo configurado.
WebSocket Twitch · YouTube · Kick Auto-remove
Dimensões recomendadas: 420 × 600 px
URL http://localhost:3000/overlay/chat
Now Playing
Exibe a música atual com capa do álbum, título e artista. Atualiza automaticamente via Windows GSMTC. Desaparece quando nada está tocando.
Polling 3s GSMTC Spotify · YT Music · ...
Dimensões recomendadas: 340 × 80 px
URL http://localhost:3000/overlay/music
Viewers
Contador de espectadores por plataforma com ponto de status animado. Cada plataforma aparece somente quando está ao vivo. Use ✏ Editar posições para arrastar os componentes livremente.
Polling 15s Twitch · YouTube · Kick Drag & Drop Auto-hide
Dimensões recomendadas: 200 × 130 px (ou livre)
URL http://localhost:3000/overlay/viewers
Camera Frame
Frame decorativo para webcam. Adicione como Browser Source no OBS sobre a fonte de vídeo da câmera. Fundo 100% transparente — apenas a borda é visível.
OBS Browser Source Transparente 13 estilos
Dimensões recomendadas: mesmo tamanho da fonte de câmera
URL http://localhost:3000/overlay/cam
Countdown
Timer de contagem regressiva para a live começar. Integrado com cenas OBS — ao terminar, troca automaticamente para a cena configurada.
OBS Auto-switch Configurável 8 temas
Dimensões recomendadas: 480 × 200 px
URL http://localhost:3000/overlay/countdown
